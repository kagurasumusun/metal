## 2. 別バージョン Xcode (`metalfe 32023.830 / .864 / .883`) 実測差分定量レポート

実機 macOS 26.4 (`sjc20-cw714-3616af18-...`) に同居する 15 種の Xcode アプリケーション (`/Applications/Xcode*.app`) に対して `xcrun metal --version` を全走査した結果、以下の 3 世代のコンパイラフロントエンド変種が観測・実証された:
1. **`metalfe-32023.830.2` (Xcode 26.0 / 26.0.1 / 26.1 / 26.1.1)**:
   - 初期世代。`-std=metal4.0` および `air64_v28` を導入した最初のコンパイラ。
2. **`metalfe-32023.864` (Xcode 26.2 / 26.2.0 / 26.3 / 26.3.0)**:
   - 中期最適化世代。`!air.compile_options` における `fast_math` オプションの細分化 (`denorms_disable` トグル) および診断文字列の整形。
3. **`metalfe-32023.883` (Xcode 26.4 / 26.4.1 / 26.5 / 26.5.0 / 26.6 / 26.6.0 / Xcode.app)**:
   - 現行マスタ世代。本プロジェクトの全 686 組み込み関数、15,643 件の `air.*` 語彙辞書、および `.metallib` (`MTLB` / `0xcbfebabe`) スライスヘッダ仕様と完全合致するリファレンスコンパイラ。
